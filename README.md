# Port-Swigger-Labs-Solved

A curated collection of solved labs from the PortSwigger Web Security Academy with detailed explanations, exploitation methodology, screenshots, payload breakdowns, HTTP requests/responses, mitigation techniques, and practical security concepts.

---

## About This Repository

This repository documents my hands-on journey through PortSwigger Web Security Academy labs to strengthen practical web application security skills and understand real-world attack vectors used in modern applications.

Each lab contains:
- Step-by-step walkthroughs
- Theory and vulnerability explanation
- Burp Suite analysis
- Payload explanations
- HTTP requests/responses
- Screenshots for proof and learning
- Mitigation techniques
- Key security takeaways

The goal of this repository is not only to solve labs, but also to deeply understand how vulnerabilities work, how attackers exploit them, and how developers can prevent them.

# Methodology Followed

For every lab:
1. Understand the application's behavior
2. Identify attack surface
3. Intercept requests using Burp Suite
4. Analyze parameters and responses
5. Craft payloads
6. Exploit the vulnerability
7. Validate successful impact
8. Document the process clearly
9. Study mitigation techniques
